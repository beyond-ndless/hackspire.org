Assembly programs require
<a href="/Ndless" class="wikilink" title="Ndless">Ndless</a> to be run. A
more up-to-date list can be found on [Ndless
Apps](https://ndless-apps.org/).

## Tools and engines

- [Ncaster](http://www.ticalc.org/archives/files/fileinfo/428/42863.html):
  Nspire Raycasting Engine
  ([forum](http://www.omnimaga.org/index.php?board=96.0))
- [Ntracer](http://www.ticalc.org/archives/files/fileinfo/426/42679.html):
  Nspire Ray Tracer demo
  ([forum](http://www.omnimaga.org/index.php?board=96.0))
- [Nleash](http://www.ticalc.org/archives/files/fileinfo/430/43033.html)
  ([forum](http://omnimaga.org/index.php?topic=3877)): Utility to bypass
  the downgrade restriction of OS 2.1. Caution, OS 1.x are *not*
  compatible with TouchPad TI-Nspire models, except with a non-CAS
  ClickPad keypad.
- [Norse](http://www.ticalc.org/archives/files/fileinfo/434/43429.html):
  Wireless chatting program, uses Press-to-Test LED to send messages in
  norse code.
- [Nover](http://www.ticalc.org/archives/files/fileinfo/434/43428.html):
  Allows one to uses the same overclocking methods used by Texas
  Instruments in OS 2.1 in earlier operating systems.
- [mViewer](http://ti.bank.free.fr/index.php?mod=archives&ac=voir&id=2014):
  Allows one to view .bmp images on the TI-Nspire.
- [nDoom](http://www.omnimaga.org/index.php?topic=5985.0): A port of the
  1997 Open Source version of the famous first-person shooter
  [Doom](http://en.wikipedia.org/wiki/Doom_(video_game)) engine
- [nPlayer](http://www.omnimaga.org/index.php?topic=6425.0): Video
  Player for the TI-Nspire
- [nwriter](http://nspforge.unsads.com/p/nwriter): Program to view or
  write/edit .txt files on the Nspire.
